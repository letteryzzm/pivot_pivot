# 三条路径题库设计（完整90题）

## 通用规则
- 每条路径30题，难度递增
- Q1-10 轻度 → Q11-20 中度 → Q21-30 高压
- 每题3-4个选项，对应四维属性：判断力(J)、行动力(A)、认知(C)、连接力(N)
- {name} 为玩家名字占位符

---

## 路径 B：想好了准备开始（行动者之路）

**Q14 - 被大厂碾压**

**Q15 - 数据造假的诱惑**
准备融资，你的数据看起来"还行"但不够"性感"。有人建议你"包装"一下数据。你会？

- B: 不造假，但选最好看的角度呈现 → J:3, A:2, C:1, N:1
- C: 数据是什么就是什么，如果不够好就继续做到够好 → J:5, C:4, A:3, N:0

**Q16 - 团队分裂**
团队里两个核心成员闹矛盾，已经影响到工作了。一个技术强但脾气差，一个协作好但效率低。你必须选一边。你会？
- A: 谁都不选，希望他们自己解决 → J:-2, A:-2, N:0, C:0
- B: 分别谈话，调解矛盾 → N:4, J:2, A:2, C:1

**Q17 - 投资人压力**

**Q18 - 核心员工被挖**

**Q19 - 身体崩溃**

**Q20 - 错误的方向**

### 第三阶段（Q21-30）：至暗时刻

**Q21 - 账上见底**

**Q23 - 投资人全部拒绝**

**Q24 - 合伙人背叛**

**Q26 - 法律危机**

## 属性维度说明

| 维度 | 含义 | 正面信号 | 负面信号 |
|------|------|---------|---------|
| 判断力 J | 独立思考，第一性原理 | 不随大流，基于数据和逻辑做决策 | 跟风，犹豫不决 |
| 行动力 A | 执行速度，动手能力 | 快速行动，先做再说 | 等待，拖延，准备过度 |
| 认知 C | 理解深度，学习能力 | 复盘，追问本质，看长期 | 表面思考，不学习 |
| 连接力 N | 关系构建，用户理解 | 主动链接，同理心，用户导向 | 封闭，自我中心 |

## 难度递增设计原则

| 阶段 | 轻度 (Q1-10) | 中度 (Q11-20) | 高压 (Q21-30) |
|------|-------------|--------------|-------------|
| 探索者 | 日常偏好和习惯 | 价值观和深层欲望 | 灵魂拷问：恐惧、牺牲、意义 |
| 准备者 | 小的时间/精力取舍 | 关系/舒适区/机会成本 | 全部押上、极端考验 |
| 已创业 | 产品和用户的日常决策 | 团队/融资/竞争的严峻考验 | 生死时刻、背叛、终极抉择 |
