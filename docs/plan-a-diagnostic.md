## 评分算法

基于文本质量分析，不看数字：

```javascript
function analyzeAnswer(answer, questionType) {
  let score = 0;

  // 1. 长度（基础分）
  if (answer.length > 100) score += 20;
  else if (answer.length > 50) score += 10;

  // 2. 关键词检查
  const keywords = getKeywords(questionType);
  keywords.high.forEach(word => {
    if (answer.includes(word)) score += 10;
  });

  // 3. 具体性
  if (hasNumbers(answer)) score += 10;
  if (hasExamples(answer)) score += 10;

  // 4. 深度
  if (hasCausalRelation(answer)) score += 10;

  return Math.min(score, 100);
}
```

---

## 优势
- 更公平：不看数字，早期项目也有机会
- 更准确：创始人质量比当前数据更重要
- 更有深度：引发创始人深度思考
- 更有价值：创始人能从问题中获得启发

## 劣势
- 需要文本分析算法
- 用户需要认真填写
- 可能筛选掉不爱写字的创始人
