## 八、使用说明

### 生成建议：
1. **优先级**：先生成核心角色形象（第一部分），再生成场景和图标
2. **风格统一**：所有图片使用相同的 "retro 8-bit pixel art" 风格
3. **尺寸规格**：
   - 角色形象：32x32px, 64x64px, 96x96px, 128x128px（像素精灵）
   - 场景背景：375x667px 或 375x667px（复古分辨率）
   - 图标：16x16px 或 24x24px（像素图标）
   - 表情包：32x32px 或 64x64px（像素表情）
4. **颜色方案**：
   - 阶段1：电蓝色 #0ea5e9
   - 阶段2：绿色 #10b981
   - 背景：黄色卡片或深灰 #18181b
   - 调色板限制：每个精灵 4-8 种颜色
5. **像素艺术要点**：
   - 使用 chunky pixel blocks（粗块像素）
   - Bold black pixel border（粗黑色像素边框）
   - Flat 2D no gradients（扁平 2D 无渐变）
   - Limited color palette（有限调色板）
   - Arcade game sprite aesthetic（街机游戏精灵美学）

### 工具推荐：
- **Midjourney**：添加 `--style raw` 参数以获得更纯粹的像素风格
- **DALL-E 3**：适合生成像素图标和简单元素
- **Stable Diffusion**：使用 pixel art 专用模型（如 Pixel Art XL）
- **Aseprite**：手动调整和优化生成的像素艺术
- **Piskel**：在线像素编辑器，用于微调

### CSS 实现提示：
```css
/* 像素艺术渲染 */
.pixel-art {
  image-rendering: pixelated;
  image-rendering: -moz-crisp-edges;
  image-rendering: crisp-edges;
}

/* 像素动画 */
.pixel-animation {
  animation: pixel-bounce 0.6s steps(3) infinite;
}

@keyframes pixel-bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-8px); }
}
```

## 十、使用说明更新

### 界面背景生成建议：

1. **尺寸规格**：所有界面背景统一使用 375x667px（iPhone X 标准尺寸）
